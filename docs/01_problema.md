# 📌 Definición del Problema

---

## 📖 Contexto

La generación de horarios académicos en la Universidad Continental representa un problema crítico de planificación, debido a la necesidad de coordinar múltiples variables interdependientes como cursos, docentes, aulas y disponibilidad horaria.

Actualmente, este proceso se realiza de manera poco optimizada, generando conflictos que afectan directamente la experiencia académica del estudiante.

---

## ⚠️ Problemática Identificada

- ❌ Solapamiento de horarios entre cursos  
- ⏳ Presencia de tiempos muertos (horas huecas)  
- 💼 Dificultad para compatibilizar estudio y trabajo  
- 🏫 Uso ineficiente de aulas  
- 📉 Aumento de la deserción estudiantil  

---

## 🧠 Complejidad del Problema

Este problema pertenece a la categoría **NP-Hard**, lo que implica que:

- No existe solución óptima en tiempo eficiente para todos los casos  
- El número de combinaciones crece exponencialmente
