# 📜 Project Charter

---

## 🎯 Objetivo del Proyecto

Desarrollar una plataforma web inteligente denominada **SmartSched-UC**, capaz de generar horarios académicos óptimos mediante el uso de técnicas de Problemas de Satisfacción de Restricciones (CSP), garantizando el cumplimiento de las normas académicas de la Universidad Continental.

---

## 📦 Alcance del Proyecto

### ✔ Incluye:
- Generación automática de horarios académicos
- Validación de prerrequisitos
- Validación del rango de créditos (20 a 25)
- Interfaz web para visualización de horarios

### ❌ No incluye:
- Integración con sistemas institucionales reales (fase futura)
- Implementación en entorno productivo real

---

## 👥 Stakeholders

| Stakeholder | Interés |
|------------|--------|
| Estudiantes | Obtener horarios flexibles |
| Docentes | Organización de clases |
| Universidad | Optimización académica |
