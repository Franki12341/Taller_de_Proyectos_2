# Evidencias de SonarQube

Contiene resultados reales de disponibilidad de SonarQube y, cuando existan credenciales de entorno, métricas autenticadas del proyecto `smartsched-uc`.

- No se almacenan tokens ni contraseñas en archivos.
- Si falta `SONAR_TOKEN`, los artefactos quedan como pendientes justificados, no como resultados falsos.
