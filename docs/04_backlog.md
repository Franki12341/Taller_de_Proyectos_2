# 📋 Product Backlog

---

## 📖 Descripción

El Product Backlog es una lista priorizada de todas las funcionalidades y actividades necesarias para el desarrollo del sistema SmartSched-UC. Este backlog se gestiona bajo la metodología ágil Scrum, permitiendo organizar el trabajo en iteraciones llamadas sprints.

Cada elemento del backlog representa una historia de usuario o tarea técnica que contribuye al desarrollo del sistema.
